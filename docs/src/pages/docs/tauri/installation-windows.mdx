# Windows Installation

This guide will help you install Jan Tauri on your Windows system.

## System Requirements

- Windows 10/11 (64-bit)
- 8GB RAM minimum (16GB recommended)
- 10GB free disk space
- Internet connection for initial setup
- DirectX 12 compatible GPU (for optimal performance)

## Installation Steps

1. **Download the Installer**
   - Visit our [GitHub releases page](https://github.com/menloresearch/jan/releases)
   - Download the latest `.exe` installer for Windows

2. **Install Jan Tauri**
   - Run the downloaded `.exe` installer
   - If prompted by Windows SmartScreen, click "More info" and then "Run anyway"
   - Follow the installation wizard
   - Choose your preferred installation location
   - Click "Install" to begin the installation

3. **First Launch**
   - Launch Jan Tauri from the Start menu or desktop shortcut
   - If prompted by Windows Defender, allow the application to run
   - The application will check your system requirements

4. **Initial Setup**
   - You'll be prompted to download your first model
   - Choose a model based on your system's capabilities
   - Wait for the model to download and initialize

## Troubleshooting

### Common Issues

1. **Windows SmartScreen Warning**
   - Click "More info"
   - Click "Run anyway"
   - This is normal for applications not from the Microsoft Store

2. **Installation Fails**
   - Ensure you have administrator privileges
   - Check if you have enough disk space
   - Try running the installer as administrator

3. **Model Download Issues**
   - Check your internet connection
   - Ensure you have enough disk space
   - Try downloading a smaller model first
   - Check Windows Defender or antivirus settings
