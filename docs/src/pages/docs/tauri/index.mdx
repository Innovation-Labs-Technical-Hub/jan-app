# Tauri Documentation

Welcome to the Jan Tauri documentation. This guide will help you understand how to use Jan with Tauri, a framework for building desktop applications using web technologies.

## Overview

Jan with Tauri provides a modern, secure, and performant way to run large language models locally on your desktop. Built with Rust and web technologies, it offers:

- **Enhanced Security**: Tauri's security-first approach ensures your AI models and data stay protected
- **Better Performance**: Native system integration and optimized resource usage
- **Smaller Bundle Size**: Significantly reduced application size compared to Electron
- **Cross-Platform**: Works on Windows, macOS, and Linux

## Getting Started

To get started with Jan Tauri:

1. Download the latest release from our [GitHub releases page](https://github.com/menloresearch/jan/releases)
2. Install the application on your system
3. Follow our [Quick Start Guide](/docs/tauri/quick-start) to begin using Jan

## Key Features

- Local LLM execution
- Model management
- Chat interface
- System resource optimization
- Cross-platform compatibility
