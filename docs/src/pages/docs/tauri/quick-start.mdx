# Quick Start Guide

This guide will help you get started with Jan Tauri quickly.

## Prerequisites

Before you begin, ensure your system meets these requirements:

- Windows 10/11, macOS 10.15+, or Linux
- 8GB RAM minimum (16GB recommended)
- 10GB free disk space
- Internet connection for initial setup

## Installation

1. Download the latest Jan Tauri release for your operating system
2. Run the installer
3. Follow the installation wizard
4. Launch Jan Tauri

## First Run

When you first launch Jan Tauri:

1. The application will check for system requirements
2. You'll be prompted to download your first model
3. Choose a model based on your system's capabilities
4. Wait for the model to download and initialize

## Basic Usage

1. **Start a New Chat**
   - Click the "New Chat" button
   - Select your preferred model
   - Begin your conversation

2. **Model Management**
   - Access the model manager from the sidebar
   - Download additional models
   - Configure model parameters

3. **Settings**
   - Adjust system resource allocation
   - Configure appearance
   - Set up API keys (optional)
