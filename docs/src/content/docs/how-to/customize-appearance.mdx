Make Jan look and feel exactly how you want with these appearance customizations.

## Change theme

<Steps>

1. **Access appearance settings**

   Go to **Settings** → **Preferences** → **Appearance & Theme**

   ![Appearance settings](/images/docs/settings-04.png)

2. **Choose your theme**

   Select from these options:
   - **Joi Light**: Clean, bright theme for daytime use
   - **Joi Dark**: Dark theme with high contrast
   - **Dark Dimmed**: Softer dark theme to reduce eye strain
   - **Night Blue**: Dark theme with blue accents

3. **Select background style** (Joi themes only)

   With Joi Dark and Joi Light, choose:
   - **Solid**: Traditional opaque background
   - **Translucent**: Semi-transparent interface

</Steps>

Changes apply immediately—no restart needed.

## Adjust chat width

Optimize your conversation view for your screen and reading preferences.

<Steps>

1. **Navigate to chat settings**

   Go to **Settings** → **Preferences** → **Chat Width**

   ![Chat width settings](/images/docs/settings-05.png)

2. **Choose your layout**

   - **Full Width**: Maximizes chat area, ideal for code and long messages
   - **Compact Width**: Focused view, better for reading on large screens

</Steps>

## Configure interface options

### Enable spell check

<Steps>

1. **Access spell check settings**

   Go to **Settings** → **Preferences** → **Spell Check**

   ![Spell check settings](/images/docs/settings-06.png)

2. **Toggle spell checking**

   - **On**: Catch typing errors in your messages
   - **Off**: Disable spell checking

</Steps>

### Show scrollbars

<Steps>

1. **Find scrollbar settings**

   Go to **Settings** → **Preferences** → **Scrolling Bar**

   ![Scrollbar settings](/images/docs/settings-19.png)

2. **Toggle visibility**

   Turn on to make scrollbars visible across all windows.

</Steps>

<Aside type="tip">
All interface changes apply immediately. Experiment with different combinations to find what works best for you.
</Aside>
