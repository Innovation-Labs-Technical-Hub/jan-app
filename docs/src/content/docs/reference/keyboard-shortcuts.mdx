Speed up your workflow with Jan's keyboard shortcuts.

<Aside type="note">
On Windows and Linux, use `Ctrl` instead of `⌘` (Command).
</Aside>

## Thread management

| Shortcut | Action |
|----------|--------|
| `⌘ N` | Create a new thread |
| `⌘ Shift Backspace` | Delete current active thread |
| `⌘ Shift C` | Clean current active thread |

## Navigation

| Shortcut | Action |
|----------|--------|
| `⌘ B` | Toggle left panel |
| `⌘ Shift B` | Toggle right panel |
| `⌘ ,` | Navigate to settings |

## Message input

| Shortcut | Action |
|----------|--------|
| `Enter` | Send a message (in input field) |
| `Shift Enter` | Insert a new line (in input field) |

## Quick Ask

| Shortcut | Action |
|----------|--------|
| `⌘ J` / `Ctrl J` | Open Quick Ask overlay |

<Aside type="tip">
Quick Ask must be enabled in **Settings** → **Advanced Settings** → **Experimental Mode**.
</Aside>
